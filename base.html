Topbar Usage

Here is an example of a topbar menu

                    <a class="topbar-menu-title">foo</a>
                    <section class="topbar-menu">
                        <a href="#blah">Link1</a>
                        <a href="#blah">Link2</a>
                        <a href="#blah">Bruh</a>
                    </section>
                    <a class="topbar-item">bar</a>

This Code will create two elements, one named "foo" with a sub-menu, and one direct link named "bar"

Note: it is mandatory to have a .topbar-menu-title just before a .topbar-menu element, or else it won't work properly

Body customization

There are some classes that you can add to the body and main elements :

body.dark

Transform the color scheme of the body (not the topbar)
to be more pleasant to the eyes

main.classical

Limit the width of the main element (basically the content block) to 60% or 1000px

SASS Customization

Of course the best way to customize your SAT setup is to edit the variables file (almost all variables are explained), but before you dive into it, let me explain the purpose of each files

  • variables.scss | the file that might interest you, it contain what you need to customize SAT
    (more documentation on this file in the file itself)
  • forms.scss | all inputs, buttons, tables...
  • generics.scss | so far, its some text properties
  • layout.scss | everythings that touch the page organization (flexboxes, spacings...)
  • root.scss | most direct properties for root, html and body elements
  • topbar.scss | only classes for topbar items and submenus
  • SAT.scss.scss | the file to compile, it is here only to contain all of the other files
